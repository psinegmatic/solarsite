«Платформа Б»

Распределённая блокчейн-платформа для хранения и обмена данными

Цели и задачи

Основная цель - разработать платформу децентрализованного хранения данных, позволяющую организовать надежный и безопасный обмен данными между конкурирующими организациями без введения в систему единого центра доверия.

Реализация

Для реализации распределенного хранилища данных была применена технология Blockchain (на базе открытой платформы Multichain). В ходе разработки наши специалисты полностью переработали, улучшили и выложили в общественный доступ (на github) библиотеки, обеспечивающие связь с платформой. Поверх блокчейна, служащего децентрализованной платформой обмена данными, реализован механизм криптографической защиты данных, позволяющий разграничить доступ к публикуемым данным. Благодаря этому механизму, построенному на базе сертифицированных в РФ средств симметричной и асимметричной криптографии, становится возможной реализация безопасной децентрализованной системы - все данные есть у всех (децентрализация), но только в зашифрованном виде, и лишь тот, для кого эти данные предназначены, может расшифровать их (безопасность)

Система реализована на платформе .NET, имеет RESTful Web API интерфейс, позволяющий внешним системам легко интегрироваться с данной платформой.

Реузльтат

Данная платформа была успешно реализована и внедрена в качестве средства обмена данными между электронными торговыми площадками и другими организациями группы компаний Финтендер.
