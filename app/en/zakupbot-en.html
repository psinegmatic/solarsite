Project goals

To provide search capability for procurements in popular messengers, such as Telegram, Skype and Facebook Messenger.

Implementation

We have used Microsoft Bot Framework to provide common code base for different communication channels. STAR search system (based on Elastic Search) has been used as data source.

Implementation process

The bot was implemented in two weeks by one developer.

Outcome

The bot has been published and works with the following messengers:

  • Telegram: @Zakupkobot
  • Skype: join.skype.com/bot
  • Facebook: www.facebook.com/Zakupkobot
