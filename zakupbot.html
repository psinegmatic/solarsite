Закупкобот

Поиска тендеров по закупкам
b2bpoint.ru

Цели и задачи

Обеспечение возможности поиска тендеров по закупкам, используя привычные каналы коммуникаций через популярные текстовые мессенджеры, такие как Telegram, Skype и Facebook Messenger.

Реализация

Технологии и инструменты

Для реализации единой кодовой базы бота, работающей с различными каналами передачи сообщений использован Microsoft Bot Framework. В качестве источника данных используется поисковая система СТАР, реализованная на базе движка Elastic Search.

Процесс реализации

Бот реализован за две недели одним разработчиком.
